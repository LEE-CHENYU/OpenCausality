# Agentic outputs

This directory stores artifacts generated by the agentic workflow.

- logs/: loop logs per agent.
- cards/data/: DataCards per NodeSpec.
- cards/models/: ModelSpecs per EdgeSpec.
- cards/results/: ResultCards/EdgeCards per edge.
- cards/judge/: system reports and edge scores.
- ledger/: spec change ledger to track iterations.
- inbox/ and outbox/: optional handoff locations between agents.
