# Agentic configuration

This folder contains configuration artifacts for the agentic DAG workflow.

- dag_schema.yaml: schema contract for DAG specifications.
- design_registry.yaml: estimator/design registry with diagnostics and templates.
- dags/: DAG instances (study-specific).
- prompts/: per-agent Codex prompt templates.
- objectives/: per-agent objective text.
- eval/: per-agent evaluation checklists.
